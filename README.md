# Графический редактор интерфейсов (PySide6)

Простое десктоп‑приложение на PySide6 для векторного рисования интерфейсов: фигуры, текст, изображения, ссылки между холстами, выравнивание и многократные холсты.

## Возможности
- Векторные фигуры: точка, кривая (свободная линия), прямая, стрелка, прямоугольник, круг, треугольник, пятиугольник, шестиугольник.
- Текстовые блоки: шрифт, размер, выравнивание (слева/центр/справа), цвет, рамка области текста.
- Изображения: загрузка внешних файлов с масштабированием по области.
- Ссылки: элементы‑ссылки для перехода на другой холст, с опциональной анимацией.
- Выделение и трансформации: множественное выделение, перемещение, изменение размера через маркеры.
- Выравнивание: по краям, по центру (гор/верт), распределение элементов по горизонтали/вертикали.
- Несколько холстов: вкладки, список холстов, переключение, закрытие.
- Сохранение/загрузка проекта в JSON (элементы и холсты).
- Подсказки (tooltip) при наведении.

## Требования
- Python 3.9+ (рекомендовано 3.10+)
- PySide6

Установка зависимостей:
```bash
pip install PySide6
```

## Запуск
```bash
python main.py
```

## Быстрый гид по интерфейсу
- Слева — панель инструментов:
  - Выделение, Точка, Кривая, Прямая, Стрелка, Прямоугольник, Круг, Треугольник, Пятиугольник, Шестиугольник, Текст, Изображение, Ссылка.
- Справа — кнопка «Выровнять» (открывает диалог выбора способа выравнивания).
- Слева в доке — список холстов, по клику переключает вкладку.
- Центральная область — холст (вкладки).

## Операции
- Рисование: выберите инструмент и тяните мышью (кривая — просто вводите мышью; текст создаётся сразу).
- Множественное выделение: инструмент «Выделение» → потяните прямоугольник.
- Перемещение: потяните выделенные элементы.
- Изменение размера: потяните маркеры по углам/ребрам выделенного элемента.
- Редактирование текста: двойной клик по текстовому блоку.
- Добавить изображение: инструмент «Изображение» → выбрать файл.
- Ссылка на холст: выберите элемент, инструмент «Ссылка» или контекстное меню «Создать ссылку…», выберите целевой холст и анимацию.
- Удаление: `Delete` (при выделении).
- Отмена выделения: `Esc`.

## Работа с проектом
- Файл → Новый холст — создаёт пустую вкладку.
- Файл → Сохранить проект — сохранить в JSON.
- Файл → Открыть проект — загрузить JSON с холстами и элементами.

## Известные моменты
- Предупреждения IDE о PySide6 можно игнорировать, если библиотека установлена.
- Несохранённые изменения не сохраняются автоматически при закрытии приложения.
